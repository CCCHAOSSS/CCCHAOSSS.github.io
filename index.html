系统特点

  • 高性能：秒杀涉及大量的并发读和并发写，因此支持高并发访问这点非常关键。
  • 数据一致性：有限数量的商品在同一时刻被很多倍的请求同时来减库存，在大并发更新的过程中都要保证数据的准确性。
  • 高可用：秒杀时会在一瞬间涌入大量的流量，为了避免系统宕机，保证高可用，需要做好流量限制。

接口和抽象类内部方法的访问权限

对抽象类来说，抽象类内部方法都是抽象方法abstract，但是抽象类中方法默认访问权限与普通类相同。如果没有指定访问修饰符，则默认为包内可见（即default访问权限）

接口内的方法默认是public，即使没有显示地声明为public。

访问权限区别：

类型 同类 同包 子类 其他包
default ✓ ✓ ✗ ✗
public ✓ ✓ ✓ ✓
