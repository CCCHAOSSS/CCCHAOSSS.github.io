接口和抽象类内部方法的访问权限

对抽象类来说，抽象类内部方法都是抽象方法abstract，但是抽象类中方法默认访问权限与普通类相同。如果没有指定访问修饰符，则默认为包内可见（即default访问权限）

接口内的方法默认是public，即使没有显示地声明为public。

访问权限区别：

类型 同类 同包 子类 其他包
default ✓ ✓ ✗ ✗
public ✓ ✓ ✓ ✓

完全背包理论基础

完全背包:有N件物品和一个最多能背重量为W的背包。第i件物品的重量是weight[i]，得到的价值是value[i] 。每件物品都有无限个（也就是可以放入背包多次），求解将哪些物品装入背包里物品价值总和最大。

和01背包问题的唯一不同是，每个物品有无限件。

0-1背包在使用滚动数组时候，物品要倒序遍历===>防止多次添加。

而完全背包不需要倒序遍历，因为可以添加多次，所以要从小到大去遍历。
